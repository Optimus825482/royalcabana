Şema Açıklaması

  • UI Katmanı: Kullanıcılar tarayıcı veya PWA üzerinden React bileşenleriyle etkileşir. shadcn/ui ve Tailwind ile stilize edilmiş.
  • Auth Katmanı: withAuth() ile korunan API rotaları, JWT ile kimlik doğrulama ve oturum yönetimini sağlar.
  • API Katmanı: Next.js App Router altında tanımlı API rotaları iş mantığını servislere yönlendirir.
  • Service Katmanı: İş kuralları, veri işleme ve dış entegrasyonları içerir.
  • Repository Katmanı: Prisma ORM üzerinden PostgreSQL ile veri erişimi sağlar.
  • Cache Katmanı: Redis, oturum ve rate‑limit gibi sık kullanılan verileri önbellekler.
  • Dış Entegrasyonlar: Open‑Meteo hava durumu, ödeme ağ geçitleri ve diğer 3rd‑party servisler.
  • Arka Plan: Socket.io gerçek‑zamanlı bildirim, Service Worker PWA offline desteği.
  • Denetim: Tüm kritik işlemler audit loglarıyla kaydedilir.

Bu şema, uygulamanın katmanlı mimarisini, veri akışını ve dış sistemlerle etkileşimini görsel olarak özetlemektedir.
